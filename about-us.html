Know About 2048 Game

2048 is a one-player sliding block puzzle game, which designed by Italian web developer Gabriele Cirulli.
The object of the game is to slide numbered tiles on a grid to match them and create a tile with the number 2048.
However, one can continue playing after reaching the goal, creating tiles with larger numbers.

How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!
